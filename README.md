# CodeDrill (Completo)

Projeto de site educacional em PHP com login, cadastro etc.

## Requisitos

- MySQL/codedrill_bd
- Servidor local (XAMPP/WAMP/Laragon) ou Apache/Nginx

## Instalação

1. Copie a pasta `CodeDrill` para seu servidor local (por ex.: `C:\xampp\htdocs\Codedrill`).

## Funcionalidades

- Cadastro e login de usuários.

## Pastas

```
Codedrill/
├── config/             # Arquivos de configuração (conexão com banco de dados)
├── functions/          # Lógica de back-end (cadastroaction.php, loginaction.php)
├── includes/           # Componentes reutilizáveis de interface (footer.php, modal.php)
├── public/             # Raiz pública do site, acessível pelo usuário
│   ├── assets/         # Subpasta para recursos estáticos
│   │   ├── css/
│   │   ├── fonts/
│   │   ├── images/
│   │   └── js/
│   ├── cadastro.php    # Página de cadastro
│   ├── index.php       # Página inicial
│   ├── login.php       # Modal de login
│   └── sobre.php       # Página "Sobre"
|   ├── cadastro.php    # Modal de Cadastro
|   ├── dashboard.php   #
|   ├──
|
```
